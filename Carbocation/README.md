Save these scripts into the final result folder that is generated from elagenteworkdir, and make sure to replace the root_dir into the corresponding root_directory for the result folder

1. Run the General_Boolean.py first, to extract the 8 key properties from the ORCA input file and ORCA output file and save into a .csv file. 

2. Run the Carbocation_specific.py, which to extract the delta-G and delta-H value in carbocations, and calculate the error rate of these values. 

3. Run the Carbocation_Score_Generator.py, which generate the overall score for this benchmarking.
